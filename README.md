# 💰 Finance Pro - Financial Management

![Finance Pro Hero](./assets/hero.png)

## 🌟 La Plataforma
**Finance Pro** es una herramienta diseñada con una estética **Sober Dark** para aquellos que buscan control total sobre sus activos con una interfaz elegante y profesional.

Desde el rastreo de inversiones hasta la gestión de presupuestos, cada detalle ha sido pulido para ofrecer una experiencia premium y minimalista.

---

## ✨ Características Principales

### 📊 Gestión de Activos
Controla tus cuentas y saldos con una interfaz intuitiva y visualmente impactante.
![Dashboard](./assets/dashboard.png)

### 🔐 Seguridad y Privacidad
Autenticación robusta y gestión de sesiones segura, protegiendo lo más importante: tus datos.
![Security](./assets/security.png)

### 🛠️ Herramientas Pro
- **Modo Demo**: Prueba la plataforma al instante con datos de ejemplo.
- **Planes Flexibles**: Diferenciación entre cuentas Free, Pro y Elite.
- **Admin Panel**: Control total para administradores del sistema.

---

## 🚀 Stack Tecnológico

| Capa | Tecnologías |
| :--- | :--- |
| **Frontend** | Next.js 14, Tailwind CSS, Framer Motion, Recharts |
| **Backend** | NestJS, Passport JWT, TypeScript |
| **Base de Datos** | SQL Server, Prisma ORM |

---

## 🛠️ Instalación

1. **Clona el repositorio**: `git clone https://github.com/elnames/Finance-Pro.git`
2. **Configura el Backend**: `cd backend && npm install && npm run start:dev`
3. **Configura el Frontend**: `cd frontend && npm install && npm run dev`
